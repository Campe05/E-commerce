99 Percent CrossFit E-commerce Website
======================================

Bienvenido al proyecto de e-commerce de 99 Percent CrossFit. Este README proporciona una visión general del proyecto, sus funcionalidades principales, y las instrucciones para su implementación y uso.

1. Descripción General
----------------------
Este proyecto es una plataforma de comercio electrónico completa para 99 Percent CrossFit, diseñada para vender equipamiento y ropa de CrossFit. La plataforma está construida utilizando React y Next.js para el frontend, y Node.js con Express para el backend.

2. Funcionalidades Principales
------------------------------
- Catálogo de productos con categorías (camisetas, sudaderas, etc.)
- Carrito de compras
- Proceso de pago seguro integrado con Stripe
- Sistema de autenticación de usuarios
- Perfiles de usuario personalizables
- Sistema de búsqueda y filtrado de productos
- Visualización 3D de productos
- Soporte multilingüe (Español e Inglés)
- Programa de fidelización con sistema de puntos
- Sistema de reseñas de productos
- Integración con Google Analytics para seguimiento avanzado
- Sistema de devoluciones de productos
